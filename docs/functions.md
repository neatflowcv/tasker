# 기능 목록

## 작업(Task) 관리

작업 관리는 데이터의 영구성을 보장하기 위해 삭제 기능을 제공하지 않습니다.
대신 더 이상 필요하지 않은 작업은 보관 처리를 통해 관리됩니다.

### 작업 생성
- 함수: `CreateTask`
- 설명: 새로운 작업을 생성합니다.

### 작업 목록 조회
- 함수: `ListTasks`
- 설명: 모든 작업 목록을 조회합니다.

### 작업 상세 조회
- 함수: `GetTask`
- 설명: 특정 ID의 작업을 조회합니다.

### 작업 보관
- 함수: `ArchiveTask`
- 설명: 작업을 보관 처리합니다.

### 작업 변경
- 설명: 작업의 변경은 기존 작업을 보관(`ArchiveTask`)하고 새로운 작업을 생성(`CreateTask`)하는 방식으로 이루어집니다. 이를 통해 작업의 변경 이력을 추적할 수 있습니다.
